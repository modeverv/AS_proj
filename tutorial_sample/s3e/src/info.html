S3E is an Amazon S3 client which allows you to manage buckets, and move files back and forth between your computer and Amazon S3. In order to use S3E, you need to sign up for the Amazon S3 web service. Once you have your S3 access key and secret access key, click on the "Account" button to enter them.

Notes about the application:

  • To upload files, click on the "Upload" button, or drag files in from other applications. Multiple files can be uploaded or dragged at the same time.
  • You can paste files in to S3E from the clipboard (only files are supported -- you can't paste other things like text).
  • To download files, your best bet is to use the "Download" button. I don't have dragging out of the application fully working right now (it's a long story). In this version, if you drag a file out, you are actually dragging out a temporary URL to the file, so you can only drag files into applications which accept URLs (like your browser). I will hopefully have better support for dragging files out in a future version, but dragging and dropping files that reside on a remote system is a difficult problem to solve.
  • S3E support keyboard copy/paste shortcuts, as well.
